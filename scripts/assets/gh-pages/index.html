Semantic UI Next CDN

This is the official CDN for Semantic UI Next. Use it to easily import Semantic UI components and utilities directly in your browser using ES modules.

Quick Start

The fastest way to get started is with an import map:

<!-- Add this to your HTML head -->
<script type="importmap" src="https://cdn.semantic-ui.com/importmap-latest.json"></script>

<!-- Then import anywhere in your code -->
<script type="module">
  import { createSignal } from '@semantic-ui/reactivity';
  import { query } from '@semantic-ui/query';
  
  // Your code here
</script>

Import Methods

1. Direct Import

Import directly with the full path including version:

import { createSignal } from 'https://cdn.semantic-ui.com/@semantic-ui/reactivity/0.10.0/src/index.js';

2. Using Import Maps (Recommended)

For a better developer experience with bare module specifiers:

<!-- Latest version -->
<script type="importmap" src="https://cdn.semantic-ui.com/importmap-latest.json"></script>

<!-- Or specific version -->
<script type="importmap" src="https://cdn.semantic-ui.com/importmap-0.10.0.json"></script>

<script type="module">
  // Now you can import without version numbers or full URLs
  import { createSignal } from '@semantic-ui/reactivity';
  
  // Your code here
</script>
