## Getting Started

You will need three things to add Semantic UI to your page.

* **Component Definition** - The component defines the CSS rules and the behaviors attached to the web component.
* **Base Theme** - The base theme adds CSS variables that are inherited across components.
* **Global CSS** (*Optional*) - Semantic UI can optionally define baseline css which will set your font stack, link colors, scrollbars and other basic global styles. For more information [see global styling](/global-styling)

<Tabs>
  <TabItem label="HTML">

    #### All Components

    Include the files in your page, loaded from a CDN or a local source.

    ```html title="index.html"
    <head>
      <!-- Defines all Web Components !-->
      <script src="https://unpkg.com/browse/@semantic-ui/core@latest/dist/semantic-ui.js"></script>
      <link rel="stylesheet" href="https://unpkg.com/browse/@semantic-ui/core@latest/dist/semantic-ui.css"></link>

      <!-- Adds Default Theme !-->
      <link rel="stylesheet" href="https://unpkg.com/browse/@semantic-ui/core@latest/dist/theme/base.css"></link>
    </head>
    <body>
      <ui-button>Button</ui-button>
    </body>
    ```

    #### Single Component
    ```html title="index.html"
    <head>
      <!-- Base global theme !-->
      <link rel="stylesheet" href="https://cdn.semantic-ui.com/themes/base/global.css"></link>
      <!-- Base theme for this component !-->
      <link rel="stylesheet" href="https://cdn.semantic-ui.com/themes/base/button.css"></link>
      <!-- Defines the component tag and makes it available on your page !-->
      <script src="https://cdn.semantic-ui.com/components/button.js"></script>
    </head>
    <body>
      <ui-button>Button</ui-button>
    </body>
    ```
  </TabItem>
  <TabItem label="Astro">


    #### HTML

    With [`@astro/lit`](https://docs.astro.build/en/guides/integrations-guide/lit/) you can import the web component and then immediately use it in an astro file.

    ```javascript title="MyPage.astro"
    import { UIButton, UIButtons } from '@semantic-ui/core';
    ```
    <aside>
      Be sure to use `client:load` for any components to allow client side javascript to run.
    </aside>
    ```javascript title="MyPage.astro"
    <UIButton client:load>Button</UIButton>
    ```

    #### Theme

    To include the base theme import the css package into your layout
    ```astro title="Layout.astro"
    ---
    import 'dist/themes/base.css' from '@semantic-ui/core`; // global theme variables
    import 'dist/semantic-ui.css' from '@semantic-ui/core`; // global css
    ---
    // rest of your layout
    ```

    To support SSR with hydration install the [lit package](https://docs.astro.build/en/guides/integrations-guide/lit/)


    ### Component Wrapper

    You could also consider loading all UI components and rendering them dynamically in a component to simplify things.

    ```astro components/UIComponent.astro
    ---
    import { UIButton, UIIcon } from '@semantic-ui/core';
    const { name, attributes } = Astro.props;
    ---

    {name === 'UIButton' && (
      <UIButton {...attributes} client:load>
        <slot />
      </UIButton>
    )}
    {name === 'UIIcon' && (
      <UIIcon {...attributes} />
    )}
    ```
