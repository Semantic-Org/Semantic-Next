## Getting Started

You will need three things to add Semantic UI to your page.

* **Component Definition** - The component defines the CSS rules and the behaviors attached to the web component.
* **Base Theme** - The base theme adds CSS variables that are inherited across components.
* **Global CSS** (*Optional*) - Semantic UI can optionally define baseline css which will set your font stack, link colors, scrollbars and other basic global styles. For more information [see global styling](/global-styling)

### All Components

<Tabs>
  <TabItem label="HTML">
    Add Semantic to your page by including a `<style>` and `<script>` tag in your page.

    ```html title="Standard HTML"

      <head>

        <!-- This defines the component tag and makes it available on your page !-->
        <script src="https://cdn.semantic-ui.com/semantic-ui.js"></script>

        <!-- This includes global base theming variables !-->
        <link rel="stylesheet" href="https://cdn.semantic-ui.com/latest/theme/base.css"></link>

        <!-- This includes global base global css (optional) !-->
        <link rel="stylesheet" href="https://cdn.semantic-ui.com/latest/semantic-ui.css"></link>
      </head>
      <body>
        <ui-button>Button</ui-button>
      </body>
    ```
  </TabItem>
</Tabs>


### Individual Components

Each UI component can be imported from the NPM package `@semantic-ui/core`. The component will automatically define itself to the corresponding camel-case tag name when imported.

For instance if you import `UIButton` you can now immediately use `<ui-button>`.

<Tabs>
  <TabItem label="HTML">
    Add Semantic components to your page by including a `<style>` and `<script>` tag in your page.
    ```html title="index.html"
      <head>
        <!-- Base global theme !-->
        <link rel="stylesheet" href="https://cdn.semantic-ui.com/themes/base/global.css"></link>
        <!-- Base theme for this component !-->
        <link rel="stylesheet" href="https://cdn.semantic-ui.com/themes/base/button.css"></link>

        <!-- Defines the component tag and makes it available on your page !-->
        <script src="https://cdn.semantic-ui.com/components/button.js"></script>
      </head>
      <body>
        <ui-button>Button</ui-button>
      </body>
    ```
  </TabItem>
  <TabItem label="Astro">

    With `@astro/lit` you can import the web component and then immediately use it in an astro file.
    <aside>
      Be sure to use `client:load` for any components to allow client side javascript to run.
    </aside>
    ```javascript title="MyPage.astro"
      import { UIButton, UIButtons } from '@semantic-ui/core';
      <UIButton client:load>Button</UIButton>
    ```
    To include the base theme you can import the css package and include in your main layout
    ~~~astro title="Layout.astro"
      ```
        import 'dist/themes/base.css' from '@semantic-ui/core`;
        import 'dist/semantic-ui.css' from '@semantic-ui/core`;
      ```
      // rest of your layout
    ~~~

  </TabItem>
