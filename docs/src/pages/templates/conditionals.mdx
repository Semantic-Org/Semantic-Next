### Control Structures

Templates support common control structures like `if`, `elseif` and `else` to handle branching paths in your template.

```handlebars
{{#if isEven number}}
  Even
{{else if isZero number}}
  Zero
{{else}}
  Odd
{{/if}}
```

### Negation

Templates include a `not` [template helper](/templates/helpers) which can be used to negate evaluated expressions


```handlebars
{{#if not isEven number}}
  Odd
{{else}}
  Even
{{/if}}
```
