## Component Templating

Components include a custom templating language built for web components. This language compiles to an Abstract Syntax Tree or `AST` that is used to render your component and surgically update portions of the DOM when reactive data sources change.

### [Expressions](templates/expressions)

Value Output
```handlebars
{{name}}
```

Function Calls
```handlebars
{{someMethod argumentOne argumentTwo}}
```

Complex Nested Expressions
```handlebars
{{formatDateTime someDate 'YYYY-MM-DD HH:mm' (object timezone='local')}}
```

Nested expressions
```handlebars
{{formatDate (getDate now) 'h:mm a'}}
```

HTML Output
```handlebars
{{{someHTML}}}
```

### Conditionals

If / elseif / else
```handlebars
{{#if isColor 'red'}}
  Red
{{else if isColor 'blue'}}
  Blue
{{else}}
  Another color
{{/if}}
```

### Loops

With intermediate value
```handlebars
{{#each row in someCollection}}
  {{row.name}}
{{/each}}
```

With inner data context
```handlebars
{{#each someCollection}}
  {{name}}
{{/each}}
```

With array index
```handlebars
{{#each someCollection}}
  {{@index}}
{{/each}}
```

### Content Slots

Default Slot
```handlebars
{{>slot}}
```
Named Slot
```handlebars
{{>slot icon}}
```

### Global Helpers


Built-in helpers
```handlebars
{{stringify someObject}}
{{formatDate someDate}}
{{log 'Hello World'}}
```

### Subtemplates

Named Templates
```handlebars
{{> exampleSubtemplate}}
```
Dynamic Template Support
```handlebars
{{>template
  name=getTemplateName
  data={
    afternoon: 'Biking',
    evening: getEveningActivity,
  }
}}
```

Reactive Data Controls
```handlebars
{{>template
  name=getTemplateName
  reactiveData={
    morning: morningActivity,
  }
  data={
    afternoon: 'Biking',
    evening: eveningActivity,
  }
}}
```

### Snippets

```handlebars
{{#snippet fullName}}
  {{#if both firstName lastName}}
    {{firstName}} {{lastName}}
  {{else}}
    {{lastName}}
  {{/if}}
{{/snippet}}

{{#if isLoggedOut}}
  Login, {{>fullName}}
{{else if hasName}}
  Login, {{>fullName}}
{{else}}
  Login
{{/if}}
