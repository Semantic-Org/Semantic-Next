Reactivity provides primitives for a signals-based reactivity system. It is used internally to power templating and rendering components, but can also be used as reactivity primitives for any custom project.

## Installation

Reactivity can be used as a standalone package or alongside [Semantic UI components](/components).

```bash
npm install @semantic-ui/reactivity
```

## Basic Usage

```javascript
import { ReactiveVar, Reaction } from '@semantic-ui/reactivity';

const counter = new ReactiveVar(0);

Reaction.create(() => {
  console.log('Counter value:', counter.get());
});

counter.set(1); // Console: "Counter value: 1"
```

## What's Inside

### Data Type Helpers

Semantic UI includes various helpers to make it easy to manipulate data like arrays, records (arrays of objects), counters, toggles, and streamline data manipulation.

* **Number Helpers** - [`increment(amount)`](/api/reactivity/number-helpers#increment), [`decrement(amount)`](/api/reactivity/number-helpers#decrement) for incrementing by amount
* **Boolean Helper** - [`toggle()`](/api/reactivity/boolean-helpers#toggle) for toggling between `false` and `true`
* **Array Mutation Helpers** - [`push()`](/api/reactivity/array-helpers#push), [`unshift()`](/api/reactivity/array-helpers#unshift), [`splice()`](/api/reactivity/array-helpers#splice), [`setIndex(index, value)`](/api/reactivity/array-helpers#setindex), [`removeIndex(index)`](/api/reactivity/array-helpers#removeindex), [`setArrayProperty(indexOrProp, property, value)`](/api/reactivity/array-helpers#setarrayproperty)
* **Record Helpers** (Array of Objects) - [`setProperty(id, property, value)`](/api/reactivity/collection-helpers#setproperty), [`replaceItem(item, id)`](/api/reactivity/collection-helpers#replaceitem), [`removeItem(id)`](/api/reactivity/collection-helpers#removeitem), [`getIndex(id)`](/api/reactivity/collection-helpers#getindex), [`getID(index)`](/api/reactivity/collection-helpers#getid), [`hasID(index)`](/api/reactivity/collection-helpers#hasid)
* **Date Helpers** - [`now()`](/api/reactivity/date-helpers#now) for setting to current time

You can learn more about these helpers in the documentation for [array helpers](/api/reactivity/array-helpers), [collection helpers](/api/reactivity/collection-helpers), [date helpers](/api/reactivity/date-helpers), and [boolean helpers](/api/reactivity/boolean-helpers).
