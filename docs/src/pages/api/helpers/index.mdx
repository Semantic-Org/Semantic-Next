## Introduction
Template helpers are used as part of [component templating](/templates) and provide access to a standard set of utilities useful for manipulating template data.

> **Templating Guide** - To learn how to use global helpers in templates see the dedicated [user guide for template helpers](/templates/helpers)

## Installation

Template helpers are part of the [templating package](/api/templating) and is a standalone package that can be used without any other dependencies.

```bash
npm install @semantic-ui/templating
```


## Usage

Template helpers can be used directly from any template


```sui
{formatDate myDate 'h:mm A'}
```
