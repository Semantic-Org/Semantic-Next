Semantic UI includes this standalone library to avoid external dependencies and to allow for performant and opinionated uses of common utils like `clone` and `isEqual` as well as some particular use cases for web components.


## Getting Started

### Installation

The utils library is a standalone package that can be used without any other dependencies.

```javascript
import { isString, clone, capitalize } from '@semantic-ui/utils';
```

### Usage

Utilities can be used by importing them in your project.

```javascript
import { isArray, capitalize, clone } from '@semantic-ui/utils';

const myArray = [1, 2, 3];
console.log(isArray(myArray)); // true

const myString = "hello world";
console.log(capitalize(myString)); // "Hello world"

const myObject = { a: 1, b: { c: 2 } };
const clonedObject = clone(myObject);
console.log(clonedObject); // { a: 1, b: { c: 2 } }
```

### Categories

The Utils library is organized into several categories:

- **[Arrays](/utils/arrays)**: Utilities for working with arrays.
- **[Browser](/utils/browser)**: Utilities for handling functionality on the client.
- **[Cloning](/utils/cloning)**: Deep cloning utilities.
- **[Crypto](/utils/crypto)**: Cryptographic helpers like hashing functions.
- **[Dates](/utils/dates)**: Helpers for handling date formats.
- **[Equality](/utils/dates)**: Helpers for comparing equality.
- **[Errors](/utils/errors)**: Error handling and creation utilities.
- **[Equality](/utils/equality)**: Deep equality comparison functions.
- **[Functions](/utils/functions)**: Helpers for manipulating functions.
- **[Looping](/utils/looping)**: Iteration utilities for arrays and objects.
- **[Numbers](/utils/looping)**: Helpers for manipulating numbers
- **[Types](/utils/types)**: Functions for checking and manipulating data types.
- **[Objects](/utils/objects)**: Helpers for object manipulation and traversal.
- **[Regular Expressions](/utils/regex)**: Helpers for working with regular expressions.
- **[SSR](/utils/ssr)**: Helpers for server-side rendering scenarios.
- **[Strings](/utils/ssr)**: String formatting and manipulation functions.
