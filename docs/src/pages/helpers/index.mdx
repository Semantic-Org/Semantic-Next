Template helpers in Semantic UI are utility functions that can be used within templates to perform various operations, from simple logical checks to complex data manipulations.

These helpers enhance the expressiveness and functionality of your templates, allowing for more dynamic and efficient rendering.

> **Tech Guide** For a guide to using global helpers in practice see the dedicated subsection for [template helpers](templates/helpers)


## Helper Categories
* **[Logical Helpers](/helpers/logical)** - Utilities for conditional checks and logical operations.
* **[String Helpers](/helpers/string)** - Functions for string manipulation and formatting.
* **[Array Helpers](/helpers/array)** - Tools for working with arrays and lists.
* **[Object Helpers](/helpers/object)** - Utilities for manipulating and transforming objects.
* **[CSS Helpers](/helpers/css)** - Functions for working with CSS classes and styles.
* **[Comparison Helpers](/helpers/comparison)** - Tools for comparing values and making decisions.
* **[Numeric Helpers](/helpers/numeric)** - Utilities for working with numbers.
* **[Date Helpers](/helpers/date)** - Functions for formatting and manipulating dates.
* **[Debug Helpers](/helpers/debug)** - Tools for debugging and logging within templates.
* **[Reactivity Helpers](/helpers/reactivity)** - Utilities for working with reactive computations.
