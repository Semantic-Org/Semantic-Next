## Key Bindings

Keyboard bindings can be attached to any component by passing in `keys` to `createComponent`

```javascript
createComponent({
  keys: {
    'ctrl + f'() {
      // handle search
    },
    'up'() {
      // handle up
    },
    'down'() {
      // handle down
    },
    'esc'() {
      // handle clear
    },
  }
});
```

A common pattern for keybinding is to invoke a behavior on the template instance
```javascript
const createComponent = () => {
  submit() {
    // submit form
  }
};
```
```javascript
const keys = {
  'enter'({tpl}) {
    tpl.submit();
  },
};
```
