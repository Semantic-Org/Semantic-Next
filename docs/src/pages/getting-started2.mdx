## Installation

To get started, install the package using npm:

```bash
npm install @semantic-ui/core
```

## Usage

<Tabs>
  <TabItem label="JavaScript">

    If you're using a module bundler like webpack, Rollup, or Parcel, you can import the components and themes directly into your JavaScript files.
    Make sure your bundler is configured to handle CSS imports.
  ```javascript
  import { UIButton } from '@semantic-ui/core';
  import '@semantic-ui/core/themes/button.css';
  ```

  </TabItem>
  <TabItem label="HTML">
  ```html title="index.html"
  <html>
  <html>
  <head>
    <link rel="stylesheet" href="https://unpkg.com/@semantic-ui/core/dist/themes/button.css">
    <script type="module" src="https://unpkg.com/@semantic-ui/core"></script>
  </head>
  <body>
    <ui-button primary>Click me</ui-button>
  </body>
  </html>
  ```
  </TabItem>

## Framework Integration

Semantic UI Web Components can be easily integrated into popular JavaScript frameworks.

<Tabs>
<TabItem label="React">
```jsx
import React from 'react';
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';

function MyComponent() {
  return <UIButton primary>Click me</UIButton>;
}
```
</TabItem>
<TabItem label="Angular">
```typescript
import { Component } from '@angular/core';
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';

@Component({
  selector: 'app-root',
  template: '<ui-button [properties]="{ primary: true }">Click me</ui-button>',
})
export class AppComponent {}
```
</TabItem>
<TabItem label="Vue">
```vue
<template>
  <UIButton :primary="true">Click me</UIButton>
</template>

<script setup>
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';
</script>
```
</TabItem>
<TabItem label="Svelte">
```html
<script>
  import { UIButton } from '@semantic-ui/core';
  import '@semantic-ui/core/themes/button.css';
</script>

<UIButton primary>Click me</UIButton>
```
</TabItem>
<TabItem label="Lit">
```javascript
import { html, LitElement } from 'lit';
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';

class MyComponent extends LitElement {
  render() {
    return html`<UIButton primary>Click me</UIButton>`;
  }
}
```

Since the underlying web components are written using Lit, you can also extend the `UIButton` component directly in your Lit components:

```javascript
import { html, LitElement } from 'lit';
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';

class MyButton extends UIButton {
  // Extend the button component
}

customElements.define('my-button', MyButton);
```
</TabItem>
<TabItem label="Astro">
```astro
---
import { UIButton } from '@semantic-ui/core';
import '@semantic-ui/core/themes/button.css';
---

<UIButton primary>Click me</UIButton>
```
</TabItem>

## Theming

Semantic UI Web Components provides pre-defined themes that you can use out of the box. However, you can also customize the components by overriding their CSS variables.

It's recommended to scope variables to :root, so they will pierce multiple shadow DOM boundaries.

```css
:root {
  --button-primary-color: var(--red);
}
```
