### How To Use This Guide

* **Adjust Sizing** - You can resize panes to adjust content size
* **Navigating** - Use next/previous below to navigate between lessons or <ui-icon menu></ui-icon> above to jump ahead.
* **Resources** - Click the resources button to see a list of related resources to any particular lesson
* **Layout** - Click the layout button to swap between code panels and tabs (this may not be available on small screen sizes).
* **AST** - You can view an AST representation of a template by clicking the "AST" tab
